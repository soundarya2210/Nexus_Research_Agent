<h1>⬡ Nexus Research — AI Web Research Agent</h1>

<p>
Nexus Research is an AI-powered research assistant that searches the web, analyzes multiple sources, 
and generates a concise research brief instantly.<br><br>

Instead of manually browsing articles, Nexus automatically collects authoritative sources 
and synthesizes them into a clear summary.<br><br>

It combines Tavily web search with Groq LLM reasoning to deliver fast, structured, 
and reliable research insights.
</p>

<h2>🚀 Features</h2>

🔎 Real-time web research <br>
📚 Multiple authoritative sources <br>
🧠 AI-generated research brief <br>
⚡ Ultra-fast Groq LLM summarization <br>
🌐 Tavily deep search integration <br>
🎨 Premium dark editorial UI <br>
🧾 Source cards with snippets <br>
📜 Query history pills <br>
📊 Structured research output <br>

<h2>📦 How to Run</h2>

<b>1. Clone the repository</b><br>

<pre>
git clone https://github.com/your-username/nexus-research.git
cd nexus-research
</pre>

<b>2. Install dependencies</b><br>

<pre>
pip install -r requirements.txt
</pre>

<b>3. Add API keys</b><br>

Create a <code>.env</code> file:<br>

<pre>
GROQ_API_KEY=your_groq_api_key
TAVILY_API_KEY=your_tavily_api_key
</pre>

<b>4. Run the app</b><br>

<pre>
streamlit run main.py
</pre>
